CARA INSTALASI

1. Unduh dan Ekstrak VibeFinder ke dalam file manager
2. Pada command prompt, alokasikan file yang sudah diekstrak
   cd C:\Users\user\Downloads\vibe-finder
3. Menginstall library yang dibutuhkan. Memasang Streamlit, Pandas, Numpy, SciKit-Learn, Plotly, dll dengan perintah
   pip install -r requirements.txt
4. Setelah instalasi selesai, jalankan aplikasi VibeFinder dengan perintah
   python -m streamlit run app.py
